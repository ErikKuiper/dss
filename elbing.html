Dutch Ships and Sailors

Dutch ships and sailors is a database that brings together maritime historical data from multiple sources to allow people to look through the data for research. The maritime datasets in this database are based on records that were kept in multiple locations including Amsterdam, Archangel, and Elbing. Most of these datasets are focused around the contracts for ships and their captains, containing information about their journey and the people who were on board. For this interface program I have selected a number the attributes found in these datasets and created a PHP script that can convert simple user input into database queries. This allows users to look through the data without having knowledge of databases or programming languages.

Elbing

The dataset Elbing is based on records of ships sailing to and from the Polish city Elbing. This dataset contains the same kind of shipdata as the other sets but it also contains interesting other information such as the date and the goods that the ships were carrying. Most of them had food but there's also other interesting things in there such as cement or pipes to smoke tobacco with. Keep in mind the data is Dutch so something like pipes will be named pijpen.

How to use the interface

The interface works by clicking on the checkbox left of the attribute that you can search for. For example clicking the checkbox left of Shipname will generate a query that searches the database for shipnames. On the right you are able to apply a filter to search for specific shipnames, for example Maria. These filters are caps-sensitive so please be specific when using them. The queries are created based on user input and selecting all the available options at once will put a lot of strain on the database so please only select the properties that you are actually interested in.
